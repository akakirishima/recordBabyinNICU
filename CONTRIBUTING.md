# Contributing

このリポジトリへの改善提案や修正は歓迎します。

## 基本方針

- 変更内容はできるだけ小さく、目的が分かる単位で提出してください
- 実データ、患者情報、識別子を含むファイルはコミットしないでください
- 環境依存の変更を行う場合は、前提 OS、Python バージョン、RealSense SDK の状況を明記してください
- スクリプトごとに仕様差分があるため、どのファイルを対象にした変更かを説明してください

## Issue

- バグ報告では、対象スクリプト名、実行環境、再現手順、期待結果、実際の結果を記載してください
- セキュリティやプライバシーに関わる情報は public issue に書かないでください

## Pull Request

- README やコメントの修正でも歓迎します
- 依存関係を変えた場合は [`requirements.txt`](/D:/hiroki/recordBabyinNICU/requirements.txt) を更新してください
- 挙動を変える修正では、何が変わるのかを PR 説明に明記してください
- 可能であれば、L515 実機で確認した結果を添えてください
