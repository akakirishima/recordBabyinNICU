# Code of Conduct

このプロジェクトでは、参加者全員が安心して議論できる場を目指します。

## Expected Behavior

- 相手を尊重した言葉づかいをする
- 技術的な意見の違いを人格攻撃に結びつけない
- 初学者を含む参加者が質問しやすい雰囲気を保つ
- 医療やプライバシーに関わる話題では、慎重で節度ある共有を行う

## Unacceptable Behavior

- ハラスメント、差別、侮辱、脅迫
- 個人情報、患者情報、機微情報の無断共有
- 破壊的行為や、他者の作業を妨害する行為

## Enforcement

問題のある振る舞いがあった場合、メンテナは注意、投稿削除、参加制限などの対応を行うことがあります。

セキュリティやプライバシーに関わる詳細は、公開の場ではなく適切な連絡方法で共有してください。
